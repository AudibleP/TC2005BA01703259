Preguntas:

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una coxion entre una n canridad de computadoras.

En cambio, WorldWideWeb es una coleccion de paginas el las cuales se navega.



¿Cuáles son las partes de un URL?

  • Protocolo: La parte de HTTP(S) se encuentra en el protocolo si tiene S al final es que es una pagina segura.
  • Dominio: Nombre utilizado para identificar las direcciones donde se encuentra.
  • Ruta: Aclara al servidos la ruta de la URL.
  • Parámetros: Filtros o datos adicionales.


¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Método que se utiliza para pedir un recurso en específico.
  • HEAD: Método que solicita los encabezados de pedir cierto recurso.
  • POST: Método que se utiiza para enviar datos en el cuerpo del request.
  • PUT: Método que se utiliza para reemplazar todas las instancias de un recurso con la carga actual.
  • PATCH: Método que se utiliza para actualizar datos almacenados en la nube.
  • DELETE: Método que se utiliza para eliminar datos en la nube.
