Preguntas Laboratorio 4:

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

La diferencia entre estos es que Java es un lenguaje compilado y el otro es un lenguaje interpretado.

¿Qué métodos tiene el objeto Date?

  • getDate(): Devuelve el día del mes. Número entre 1 y 31
  • getDay(): Devuelve el día de la semana. Entre 0 (domingo) y 6 (sábado)
  • getFullYear(): Devuelve el año con 4 dígitos
  • getMilliseconds(): Devuelve los milisegundos entre 0 y 9999
  • getMinutes(): Devuelve los minutos. Entre 0 y 59

¿Qué métodos tienen los arreglos?

  • map(): Permite recorrer el array y modificar los elementos presentes en él, retornando un nuevo array con la misma longitud que el original.
  • filter(): Recorre el array y retorna un nuevo array con aquellos elementos que pasen una determinada condición.
  • forEach(): Permite iterar el contenido de un array. Recibe un callback que toma como parámetro el elemento actual de la iteración y el indice del mismo.
  • find(): Recorre el array y retorna la primera coincidencia del elemento que se busca.
  • sort(): Ordena los elementos del array y retorna el arreglo ordenado. Los elementos se ordenarán en orden ascendente (de la A a la Z) por defecto.

¿Cómo se declara una variable con alcance local dentro de una función?

let es una palabra reservada que se utiliza para declarar variables locales.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

var dentro de la función, se crea como una variable local, pero esto permite que el usuario maneje esta variable a su gusto.
